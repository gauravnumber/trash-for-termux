# trash-for-termux

This is shell based script, no need of python or node, no external dependencies required.

## Flags

Show size of trash

```
trash -s
```

Show all files in trash

```
trash -l
```

Restore all files.
You have give where you want to store all files

```
trash -r <path>
```

Empty all files in trash

```
trash -e
```
